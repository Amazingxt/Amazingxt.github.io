遇到的问题

  • 办公室里的电脑一直装不上npm，试过用淘宝镜像等方法，没有作用，最后将其部署到笔记本上，可用
  • 在github上创建Repository的时候，尽量将名字定义成与github的用户名保持一致（在coding上同样），这样可以直接通过username.github.io进行访问，否则博客内容就会进入下一级目录username.github.io/Reponame.github.io
