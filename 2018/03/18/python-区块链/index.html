区块链是由被称为区块的记录构成的不可变的、有序的链式结构，这些记录可以是交易、文件或任何你想要的数据，最重要的是它们是通过 Hash 连接起来的。—–引用自博客

简单的解释就是Hash函数相当于一个黑箱子，每一个input对应着唯一的一个output，你可以将input输入进盒子得到output，但是没有办法通过output得到input。Hash很大的一个作用就是你可以在不用知道输入是什么的时候进行验证。
